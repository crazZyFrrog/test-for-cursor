# Список дел на день

Простое веб-приложение для управления задачами на день.

## Возможности

- ✅ Добавление новых задач
- ✅ Отметка задач как выполненных
- ✅ Удаление задач
- ✅ Статистика (всего, выполнено, осталось)
- ✅ Очистка всех выполненных задач
- ✅ Автоматическое сохранение в браузере (localStorage)
- ✅ Отображение текущей даты

## Запуск

### Способ 1: Через локальный сервер (рекомендуется)

**Вариант А - Двойной клик:**
- Дважды кликните на файл `start-server.bat`
- Браузер откроется автоматически

**Вариант Б - Через терминал:**
```bash
python start-server.py
```
Затем откройте в браузере: http://localhost:8000/index.html

### Способ 2: Прямой запуск в браузере

- Найдите файл `index.html` в проводнике Windows
- Дважды кликните по нему
- Файл откроется в браузере по умолчанию

### Способ 3: Через PowerShell

```powershell
cd "c:\Users\1\Desktop\curse\vibe coding\test-for-cursor"
start index.html
```

### Способ 4: Через Cursor (если встроенный браузер не работает)

1. Откройте файл `index.html` в редакторе
2. Нажмите `Ctrl+Shift+P`
3. Введите "Simple Browser" или используйте расширение "Live Preview"
4. Или просто скопируйте полный путь к файлу и вставьте в адресную строку любого браузера

## Использование

- Введите задачу в поле ввода и нажмите Enter или кнопку "Добавить"
- Отметьте задачу как выполненную, используя чекбокс
- Удалите задачу, нажав кнопку "Удалить"
- Очистите все выполненные задачи одной кнопкой

Все данные сохраняются автоматически в браузере и будут доступны при следующем открытии приложения.
