# Инструкция по отправке изменений в GitHub

## Быстрый способ (через терминал Cursor)

### 1. Добавить все изменения
```bash
git add .
```

### 2. Создать коммит с описанием
```bash
git commit -m "Описание ваших изменений"
```

Примеры сообщений коммитов:
- `"Добавлена темная тема"`
- `"Обновлен дизайн списка дел"`
- `"Исправлены ошибки в стилях"`

### 3. Отправить в GitHub
```bash
git push origin main
```

## Полный процесс (пошагово)

### Шаг 1: Проверить статус
```bash
git status
```
Покажет, какие файлы изменены.

### Шаг 2: Добавить файлы
```bash
# Добавить все файлы
git add .

# Или добавить конкретные файлы
git add style.css index.html
```

### Шаг 3: Создать коммит
```bash
git commit -m "Ваше описание изменений"
```

### Шаг 4: Отправить в GitHub
```bash
git push origin main
```

## Если возникли проблемы

### Если нужно отменить изменения в файле
```bash
git checkout -- имя_файла.css
```

### Если нужно отменить коммит (но оставить изменения)
```bash
git reset --soft HEAD~1
```

### Если нужно посмотреть историю коммитов
```bash
git log --oneline
```

### Если нужно обновить локальный репозиторий из GitHub
```bash
git pull origin main
```

## Через интерфейс Cursor (GUI)

1. Откройте панель Source Control (Ctrl+Shift+G)
2. Нажмите "+" рядом с измененными файлами, чтобы добавить их
3. Введите сообщение коммита в поле сверху
4. Нажмите кнопку "Commit" (галочка)
5. Нажмите "Sync Changes" или "Push" для отправки в GitHub
